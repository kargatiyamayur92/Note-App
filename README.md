# 📝 Notes App

A modern Full Stack Notes Application built with **React.js, Node.js, Express.js, and MongoDB**. It allows users to securely manage notes with authentication, categories, themes, profile management, PDF export, and many productivity features.

---

## 🚀 Features

### 🔐 Authentication

* User Registration
* User Login
* JWT Authentication
* Protected Routes
* Password Encryption using bcrypt

### 📝 Notes Management

* Create Notes
* Read Notes
* Update Notes
* Delete Notes
* Permanent Delete
* Restore Deleted Notes

### 📌 Productivity

* Pin / Unpin Notes
* Search Notes
* Filter by Category
* Sort Notes (A-Z, Z-A, Latest, Oldest)
* Change Note Background Color
* Export Notes as PDF

### 🗂 Categories

* Study
* Work
* Personal
* Ideas

### 📊 Dashboard

* Total Notes
* Pinned Notes
* Category Statistics
* Pie Chart Visualization

### 🗑 Trash

* Restore Notes
* Delete Permanently
* Delete All Trash

### 👤 User Profile

* Change Profile Picture
* Update Username
* Change Password

### 🎨 Customization

* Dark Mode
* Light Mode
* Multiple Color Themes

### 📱 Responsive Design

* Mobile Friendly
* Tablet Friendly
* Desktop Friendly
